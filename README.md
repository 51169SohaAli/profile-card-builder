# Profile Card Builder
A frontend task built as part of a technical interview

# Brief Description
This profile card builder allows the user to enter inputs and upon clicking the submit button, they can see what their profile card looks like.

# How to run 
Open index.html directly in browser
For React part: use this code to run with vite:
[cd react
npm install
npm run dev]

# Tech Used
HTML, CSS, JavaScript and React

# Time Taken
~58 minutes
